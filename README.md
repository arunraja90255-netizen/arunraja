# Vulnerability Scanner Mini Project

## Overview
This project is a web-based vulnerability scanner developed using Python and Flask.

## Features
- Open Port Detection
- Weak Configuration Detection
- Vulnerability Reporting
- Simple Web Interface

## Technologies Used
- Python
- Flask
- Nmap
- HTML
- CSS

## Project Workflow

User Input
↓
Port Scan
↓
Vulnerability Analysis
↓
Report Generation

## Sample Results

Open Ports:
- 22 SSH
- 80 HTTP
- 443 HTTPS

Risk Level: Low
